## Oldtimers Offer is a Web3-powered classic vehicles marketplace combining Rust performance, WordPress SEO, and XRPL-based NFT vehicle passports.


Oldtimers Offer is a Web3-powered classic vehicles marketplace designed to deliver a modern, high-performance experience by combining Rust (Actix Web backend and Yew WebAssembly frontend) with WordPress for SEO and content management. The platform allows users to browse, list, and manage classic vehicles, while integrating XRPL-based NFT vehicle passports that provide verifiable, on-chain representations of vehicle data and ownership. Built on a modular microservices architecture, the system includes dedicated services for image upload and optimization, NFT minting, and cloud-based storage with CDN delivery. With a strong focus on scalability, performance, and cross-platform availability (web, Android, and Microsoft Store), Oldtimers Offer bridges traditional marketplaces with Web3 infrastructure to deliver a fast, secure, and future-ready user experience.
